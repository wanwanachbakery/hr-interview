# Job Description (ฉบับร่างจาก workflow จริง)

## ตำแหน่ง
**IT Manager** — แผนกIT

## สรุปบทบาท (Role Summary)
ดูแลทีมฝ่ายไอที 4-5 คน คอย Support งานด้านซอฟต์แวร ฮาร์ดแวร์ และงานขายหน้าร้าน เช่น ระบบ POS, Network ด้านงาน Head Office ดูและระบบ ERP D365 Business Central รวมทั้งเมล์ MS-O365

## หน้าที่หลัก (Primary Responsibilities)
- เวลา 10:00 เช็ครายงานจากทีมผ่านระบบ MS Teams
- ช่วงเวลา 11:00 เข้าตรวจสอบระบบ JC-Global on Production ว่ามี bug หรือ  session error ในการดึงข้อมูลไหม
- ช่วงเวลา 12:00 เริ่ม Dev งาน ai ระบบ JC-Market ในส่วนการดึงข้อมูล  API จาก BC มาแสดงในรายการการสินค้า
- ช่วงเวลา 13:00 Dev งาน ai ระบบ JC-Market ต่อ แต่เป็นส่วนของ ส่งข้อมูล API เข้า BC แล้วสร้างเอกสาร SO
- ช่วง 15:00 ทดสอบผลลัพธ์ของงาน JC-Market ว่าได้ผลลัพธ์ถูกต้องไหม
- ช่วง 16:00 แก้ไขงานระบบ JC-Market
- ช่วง 17:00 ประชุมงานพัฒนา API เชื่อมต่อระบบ ERP TH กับ KindDee ERP CN โดยมีคุณเติ้ลเป็น Lead
- ช่วง 19:00 วาง Flow การพัฒนา API เชื่อมต่อระบบ ERP TH กับ KindDee ERP CN กับทีมไอทีเจี้ยนชา เพื่อนำไปหารือกับผู้ให้บริการ ERP ประเทศไทย คือคุณเก่งจาก บ Exsys
