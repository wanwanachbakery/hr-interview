# Job Description (ฉบับร่างจาก workflow จริง)

## ตำแหน่ง
**HROD ,TA Supervisor** — แผนก

## สรุปบทบาท (Role Summary)
รับผิดชอบงานประจำวันในแผนก โดยเน้นการถ้าเป็นวันอังคาร-ศุกร์จะโทรคุยกับผู้สมัครหลังจากนั้นส่งข้อมูลผู้สมัครที่ผ่านการสัมภาษณ์เบื้องต้นให้กับทางต้นสังกัดพิจรณาต่อ
และประสานงานร่วมกับทีมHR ทั้งหมด ทีมออฟฟิศทั้งหมดเช่นBD, SCM, MKT เพื่อสนับสนุนเป้าหมายของบริษัท

## หน้าที่หลัก (Primary Responsibilities)
- ถ้าเป็นวันอังคาร-ศุกร์จะโทรคุยกับผู้สมัครหลังจากนั้นส่งข้อมูลผู้สมัครที่ผ่านการสัมภาษณ์เบื้องต้นให้กับทางต้นสังกัดพิจรณาต่อ
- นัดสัมภาษณ์ในกรณีที่ผู้สมัครผ่านการพิจารณา ส่วนวันจันทร์จะตรวจเช็คความคืบหน้างานสรรหาของทีมเพื่อทำรีพอตส่งหัวหน้า
- ผู้บริหารในการประชุมทุกวันอังคาร
- มีประชุมทุกวันพฤหัสบดี ส่วนวันอื่นๆจะใช้ทำงานส่วนอื่นนอกจากงานสรรหาเช่นงานประชุมกับแผนกอื่นๆเรื่องขั้นตอนการทำงานเพื่อเขียนโครงสร้างองค์กรหรืองานคิดนโยบายสวัสดิการต่างๆการพัฒนาองย์กร หรือถ้าเป็นงานสรรหาก็จะเป็นการObserveสัมภาษณ์
- ประเมินผู้สมัคร

## หน้าที่รอง (Secondary Responsibilities)
- มีสรุปงานของทีมเพื่อใช้สั่งการในวันถัดไปที่ต้องทำทุกวัน
- งานทำรีพอตทุกวันจันทร์กับงานทำรีพอตประชุมทุกวันพฤหัสบดี
- สรุปงานเพื่อเตรียมวันพรุ่งนี้หรือหากมีงานด่วนก็จะสรุปแล้วกลับไปทำที่บ้าน
- ทุกวันจันทร์ทำรีพอตเพื่อเตรียมประชุมวันอังคาร
- ทุกวันพฤหัสบดีเช้าทำรีพอตเพื่อเตรียมประชุมวันพฤหัสบดีบ่าย กับทำเอกสารประเมินProbation ของหน้าสาขาต้องทำภายในวันที่ 21 ของทุกเดือนส่วนของสำนักงานใหญ่ต้องทำให้เสร็จก่อนวันที่ 29-30 ของเดือน
