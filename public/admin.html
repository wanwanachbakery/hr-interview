🛑 Danger Zone — ลบข้อมูล

แต่ละปุ่มจะถามยืนยันก่อนลบ · ลบแล้วกู้ไม่ได้ (ใช้ 3-BACKUP.bat สำรองข้อมูลก่อน)

1️⃣ ลบ User ทั้งหมด
เคลียร์ user รายคน · emp records ถูก archive · เก็บ ฝ่าย/แผนก/ตำแหน่ง + ประวัติ interview
2️⃣ ลบ ฝ่าย/แผนก/ตำแหน่ง
เคลียร์โครงสร้างองค์กร + cascade ลบ user + emp + interviews ด้วย (เพราะอ้างอิงกัน)
3️⃣ ลบข้อมูลที่ User กรอก
เคลียร์เฉพาะคำตอบ interview + เอกสาร JD/KPI · reset emp status เป็น not_started · เก็บ user + org
4️⃣ ลบข้อมูลทั้งหมด
Factory reset · เคลียร์ทุกอย่าง · เก็บแค่ admin password
