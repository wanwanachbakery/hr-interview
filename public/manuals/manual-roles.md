# คู่มือการใช้งานระบบ HR-Interview (ฉบับตามบทบาท)

คู่มือฉบับนี้สรุป "สิ่งที่ทำได้" และ "ขั้นตอนงานหลัก" ของแต่ละบทบาท เพื่อให้ผู้ใช้แต่ละคนเปิดอ่านเฉพาะส่วนของตนได้รวดเร็ว หากต้องการรายละเอียดทุกฟังก์ชัน ดูที่ "คู่มือฉบับเต็ม"

---

## ภาพรวมบทบาท

| บทบาท | เห็นข้อมูล | งานหลัก |
|-------|-----------|---------|
| เจ้าหน้าที่ (officer) | เฉพาะของตนเอง | ตอบสัมภาษณ์ + บันทึกงานประจำวัน |
| หัวหน้าแผนก (section_head) | เฉพาะแผนกของตน | งานของตน + ดูบันทึก/สถานะของลูกน้องในแผนก |
| หัวหน้าฝ่าย (division_head) | เฉพาะฝ่ายของตน | เหมือนหัวหน้าแผนก แต่ครอบคลุมทั้งฝ่าย |
| ผู้จัดการ (manager) | ฝ่ายของตน + ที่ได้รับอนุญาตเพิ่ม | ดูแลทีม + วิเคราะห์ภาพรวม |
| ผู้บริหาร (executive) | ทั้งบริษัท (อ่าน) | ดูภาพรวมทั้งองค์กร |
| ผู้ดูแลระบบ (admin) | ทั้งบริษัท + ตั้งค่า | จัดการทุกอย่างในบริษัท |
| ผู้ดูแลระดับสูง (super admin) | ระดับหลายบริษัท | สร้าง/ลบบริษัท |

ทุกคนเริ่มต้นเหมือนกันคือ **เข้าสู่ระบบ** ด้วย username/รหัสผ่านที่ได้รับ และ **ออกจากระบบ** ทุกครั้งเมื่อใช้เสร็จ (ปุ่มมุมขวาบน)

---

## 1. เจ้าหน้าที่ (Officer)

**เห็นอะไร:** ข้อมูลของตนเองเท่านั้น
**เมนูที่ใช้:** หน้าแรก, บันทึกงาน, Dashboard, รายงาน, คู่มือ

### งานหลัก ก: ตอบการสัมภาษณ์งานของตน
1. ที่หน้าแรกหรือ Dashboard กดเริ่ม/ตอบการสัมภาษณ์ของตน
2. ตอบคำถามตามช่วงเวลา (งานที่ทำ เครื่องมือ คนที่ประสาน ปัญหา ฯลฯ)
3. กด "เสร็จสิ้น" รอจนขึ้น "วิเคราะห์สำเร็จ"
4. เปิดดูเอกสาร 6 ฉบับที่ระบบสร้างให้ได้

### งานหลัก ข: บันทึกงานประจำวัน
1. เข้าเมนู "บันทึกงาน" เลือกวันที่
2. พิมพ์งานในแต่ละชั่วโมง เลือกหมวดหมู่
3. กด "+ เพิ่มงานในชั่วโมงนี้" หากมีหลายงานในชั่วโมงเดียว
4. ติ๊ก "งานประจำ" สำหรับงานที่ทำซ้ำทุกวัน (วันถัดไปกด "คัดลอกงานประจำจากเมื่อวาน" ได้)
5. กด "บันทึกงานวันนี้"

> ควรบันทึกงานทุกวัน เพราะข้อมูลนี้จะช่วยให้เอกสาร KPI/Optimization ของคุณแม่นยำขึ้น

---

## 2. หัวหน้าแผนก (Section Head)

**เห็นอะไร:** ข้อมูลของแผนกตนเอง (ตนเอง + ลูกน้องในแผนก)
**เมนูที่ใช้:** ทุกเมนูของเจ้าหน้าที่ + บันทึกงานทีม + สรุปบันทึกงาน

### สิ่งที่ทำได้
- ตอบสัมภาษณ์ + บันทึกงานประจำวันของตน (เหมือนเจ้าหน้าที่)
- ดูสถานะการสัมภาษณ์ของลูกน้องในแผนกผ่าน Dashboard
- ดูบันทึกงานประจำวันของลูกน้องผ่านเมนู "บันทึกงานทีม"
- ดูรายงานสรุปบันทึกงานของแผนกผ่าน "สรุปบันทึกงาน"

### ขั้นตอน: ดูบันทึกงานของลูกน้อง
1. เข้าเมนู "บันทึกงานทีม"
2. เลือกพนักงานจากดรอปดาวน์ (พิมพ์ชื่อเพื่อค้นหา) หรือคัดกรองตามแผนก
3. เลือกวันที่ ระบบแสดงตารางงานรายชั่วโมง (อ่านอย่างเดียว)

---

## 3. หัวหน้าฝ่าย (Division Head)

**เห็นอะไร:** ข้อมูลทั้งฝ่ายของตน (ทุกแผนกในฝ่าย)
**เมนูที่ใช้:** เหมือนหัวหน้าแผนก แต่ขอบเขตกว้างเป็นทั้งฝ่าย

### สิ่งที่ทำได้
- ทุกอย่างเหมือนหัวหน้าแผนก แต่เห็นทุกแผนก/ทุกตำแหน่งในฝ่ายของตน
- ใช้ "บันทึกงานทีม" และ "สรุปบันทึกงาน" เพื่อติดตามทั้งฝ่าย
- ใช้ Dashboard ดูสถานะการสัมภาษณ์ของทั้งฝ่าย

---

## 4. ผู้จัดการ (Manager)

**เห็นอะไร:** ฝ่ายของตน และฝ่ายที่ผู้ดูแลอนุญาตเพิ่มเติม
**เมนูที่ใช้:** เหมือนหัวหน้าฝ่าย + สิทธิ์วิเคราะห์ภาพรวม

### สิ่งที่ทำได้
- ดูแลและติดตามทีมในขอบเขตของตน (Dashboard, บันทึกงานทีม, สรุปบันทึกงาน, รายงาน)
- **วิเคราะห์ภาพรวมบริษัท:** กดปุ่ม "วิเคราะห์ภาพรวมบริษัท" ใน Dashboard เพื่อสร้างรายงาน Optimization รวม

### ขั้นตอน: สร้างรายงานภาพรวม
1. เข้า Dashboard
2. กดปุ่ม "วิเคราะห์ภาพรวมบริษัท"
3. รอจนเสร็จ แล้วอ่านรายงานในหน้า Dashboard หรือกด "เปิดรายงาน" เพื่อดาวน์โหลด

---

## 5. ผู้บริหาร (Executive)

**เห็นอะไร:** ข้อมูลทั้งบริษัท (อ่าน)
**เมนูที่ใช้:** Dashboard, รายงาน, บันทึกงานทีม, สรุปบันทึกงาน, วิเคราะห์ภาพรวม

### สิ่งที่ทำได้
- ดูสถานะการสัมภาษณ์และเอกสารของทุกคนทั้งบริษัท
- ดูบันทึกงานและรายงานสรุปของทุกทีม
- สร้าง/อ่านรายงานวิเคราะห์ภาพรวมองค์กร

> ผู้บริหารเน้นการ "ดูภาพรวมเพื่อการตัดสินใจ" การจัดการผู้ใช้/โครงสร้างเป็นหน้าที่ของผู้ดูแลระบบ

---

## 6. ผู้ดูแลระบบ (Admin)

**เห็นอะไร:** ทุกอย่างในบริษัท + ตั้งค่าระบบ
**เข้าระบบ:** username `admin` + รหัส Master Admin

### สิ่งที่ทำได้ (สรุป)
- จัดการโครงสร้างองค์กร: ฝ่าย/แผนก/ตำแหน่ง
- จัดการผู้ใช้: เพิ่ม/แก้ไข/ลบ กำหนดบทบาทและสังกัด
- นำเข้าข้อมูลด้วย Excel (โครงสร้าง + ผู้ใช้)
- เปลี่ยนรหัส Master Admin
- ตั้งค่า Claude AI: เปิด/ปิด (ยืนยันด้วยรหัส Admin) และดูค่าใช้จ่าย/โทเคน
- รีเซ็ตการสัมภาษณ์ของพนักงาน (กลับเป็น "ยังไม่ตอบ")
- ลบข้อมูลเป็นกลุ่มใน Danger Zone

### ลำดับการตั้งค่าระบบใหม่ (แนะนำ)
1. เปลี่ยนรหัส Master Admin
2. สร้างฝ่าย → แผนก → ตำแหน่ง (หรือใช้ Excel นำเข้า)
3. สร้างผู้ใช้แล้วผูกกับตำแหน่ง (หรือใช้ Excel นำเข้า)
4. ตั้งหมวดหมู่งานเริ่มต้นสำหรับบันทึกงานประจำวัน
5. (ถ้าต้องการ) ตั้งกุญแจ API และเปิดสวิตช์ Claude

### ข้อควรระวังสำหรับผู้ดูแล
- สำรองข้อมูลก่อนใช้ Danger Zone ทุกครั้ง
- การเปิด/ปิด Claude ต้องกรอกรหัส Admin ยืนยัน
- ปุ่ม "รีเซ็ต" ล้างคำตอบ/เอกสารแต่ไม่ลบพนักงาน

---

## 7. ผู้ดูแลระดับสูง (Super Admin)

**เห็นอะไร:** ระดับหลายบริษัท (ไม่ยุ่งข้อมูลภายในบริษัท)
**เข้าระบบ:** `https://ชื่อเว็บ/super/login` ด้วยรหัส Super Admin

### สิ่งที่ทำได้
- สร้างบริษัทใหม่ (กำหนดรหัสบริษัท ชื่อบริษัท รหัส Admin เริ่มต้น)
- ดูรายชื่อบริษัททั้งหมด
- ลบบริษัท

### ขั้นตอน: สร้างบริษัทใหม่
1. เข้าสู่ระบบที่หน้า Super Admin
2. กดสร้างบริษัทใหม่
3. กรอกรหัสบริษัท (ใช้ในลิงก์), ชื่อบริษัท, และรหัส Admin เริ่มต้น
4. ส่งลิงก์เข้าระบบและรหัส Admin ให้ผู้ดูแลของบริษัทนั้น เพื่อให้เริ่มตั้งค่าต่อ
